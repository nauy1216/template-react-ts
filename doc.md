# 1、创建项目

```
create-react-app my-app --typescript
```



# 2、启动项目

```
npm start
```

启动后打开浏览器，报错：

```
 ReactRefreshEntry.js:8 报错 Cannot read property 'forEach' of undefined
```

原因是`react-dev-tool`导致的报错，关闭即可。
